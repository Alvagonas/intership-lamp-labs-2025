Описание:
backend-сервис на Java с базой данных и API

Функциональность:
POST /api/notes — создать заметку (получает JSON: title, content)
GET /api/notes/{id} — получить заметку по ID
GET /api/notes — получить список всех заметок

Технический стек:
Java 21
Spring Boot 3.5.0
Spring Data JPA
PostgreSQL 13
Docker
